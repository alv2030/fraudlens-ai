# Feature Engineering

FraudLens AI focuses on behavioral and velocity features rather than raw transaction fields only.

Key features:

- amount_to_avg_ratio
- transaction_hour
- is_night_transaction
- is_foreign_transaction
- is_new_device
- is_card_not_present
- velocity_1h
- velocity_24h
- merchant_risk_score
- customer_avg_amount_30d

These features mimic real fraud detection patterns such as abnormal spending, unusual location, suspicious devices, and repeated transaction attempts.
